#  笑着活下去
:monkey_face: 狂奔的男尸  :clock1: 2017-08-27 20:36:46  :open_file_folder:   文艺创作

和前女友认识第二个星期去开房。

还是我的第一次。

前台一问，双标和大床房原来是一个价，迫不及待地我说那当然要大床啦。

前女友说，不，要双标。

后来进到房子里，她才说我不懂这里面一个细节，那就是，双标的两张床并非两张单人床，而是和大床差不多的两个人都能睡的床。

我还是不明白：这算什么细节呀？

她说：这你就不懂了吧？大床搞湿了以后睡起来很难受，而双标搞湿一张床还可以舒舒服服地睡另一张床。

多年以后的今天，每当想到这个细节，我都会笑着活下去。
